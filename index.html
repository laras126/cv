Lara Schenck

notlaura.com
Prospective Ph.D. Student

Industry leader in web development seeking to transition from software engineering to computing education research. Research interests include:

  • Notional machines for web development
  • Unplugged CS instructional methods for adult learners
  • Broadening participation in computing
  • lara@notlaura.com
  • (724) 244-2491
  • Pittsburgh, PA
  • Github
  • LinkedIn

Research

  • CSS Algorithms – Independent

    2017 - Present

    Independent research into what is a programming language vs. what developers think is a programming language. Invented the concept of CSS algorithms to promote inclusive understanding of programming.

    • Paper (unpublished): Is C more of a Programming Language that Python? Exploring Cognitive Bias In Programming Language Perception – Collaboration with Felienne Hermans to report results of an experimental survey.
    • Conference Speaking: CSS Algorithms presented at 10 conferences internationally from 2018-2019.
    • Workshop: Generative Art With FizzBuzz in CSS – Designed 1 hour workshop for intermediate-level high school students attending Code Day 2021 online. Developed pre-post survey to evaluate effectiveness of learning objectives.
    • Tutorials & Articles: CSS Is A Declarative, Domain-Specific Programming Language, Is CSS Turing Complete?, Writing CSS Algorithms

  • Larva Design System – Penske Media Corporation

    2018 - Present

    At Penske Media Corporation, conceived of and implemented a front-end UI framework (design system) to speed up front-end development and reduce technical debt. Implemented the concept of CSS algorithms. This page is built with Larva.

    • Conducted need-finding interviews with engineers, designers, and product managers
    • Presented design system concepts and value proposition to executives; got buy-in
    • Created and tested prototypes to inform architecture
    • Implemented CSS algorithms as a core piece of the CSS architecture along with utility-forward CSS library
    • Conducted monthly "Larva Chats" with engineering and product for knowledge sharing and feedback collection
    • Outlined development of design system in informal blog post series entitled Designgineering Chronicles
    • Created proof-of-concept for visual regression testing; currently working to automate it
    • Devised and developed tools for product and designers to communicate more effectively with engineering using the system nomenclature

Experience

  • Senior Software Engineer – Penske Media Corporation

    2018 - Present

    Full stack web engineering for high-traffic, enterprise WordPress websites. Successfully led re-platform and upgrade initiatives across a network of 25 sites. Advocate for equity and inclusion in team culture, hiring, and compensation.

  • Independent Consultant – Lara Schenck, LLC

    2011 - 2017

    Contract front-end and WordPress development and training for startups and small to large businesses and web development agencies.

  • Web Development Instructor

    2013 - 2018

    Developed curriculum and instructed web development in the forms of one-on-one tutoring, independently organized courses, and for a variety of institutions.

    • Pratt Institute of Design – UI/UX Prototyping in Code, Coding I-II
    • University of Southern California – WordPress Development
    • Tacklebox School – Independently organized online courses in WordPress and web development, recorded and published online
    • Girl Develop It – Hands-on Sass Fundamentals, Coding for WordPress, Building Custom WordPress Websites and Intro to HTML/CSS
    • Noble Desktop – WordPress.org
    • Decoded – Code in A Day Facilitator
    • General Assembly – Front-end Web Development
    • CSS-Tricks – "Office Hours" instruction program organizer
    • Saxifrage School – Intro to Web Development

  • Design Assistant – Institute of Play

    2013 - 2014

    Support for the development of game-like curricular materials in the Mission Lab at Quest to Learn, a NYC public middle and high school that focuses on systems thinking and game-like learning. Collaborate with team of game designers and educators to produce digital and non-digital game assets.

Volunteer

  • Technical Advisor & Web Developer – Supreme Network Global

    2021 - Present

    Provide technical guidance and education for founder and returning citizen Andre Jacobs as he launches Supreme Network, a membership company that empowers inmates and returning citizens with mentorship, education, and financial opportunities.

  • Open Source Contributor – WordPress Core

    Facility weekly meetings of the Core CSS team, devise and oversee and contribute code to long term projects improving the health and maintainability of the WordPress Core CSS code-base.
